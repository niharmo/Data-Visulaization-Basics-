# Data Visualization with Matplotlib and Seaborn

This project uses the Tips Dataset from Seaborn to practice creating different types of charts using both Matplotlib and Seaborn. The goal is to understand the syntax, customization options, and differences between the two visualization libraries in Python.

## Project Requirements

* Load the dataset using `sns.load_dataset("tips")`
* Create charts using both Matplotlib and Seaborn:

  * Histogram
  * Bar Chart
  * Box Plot
  * Scatter Plot
  * Heatmap of Correlations
* Customize:

  * Titles
  * Labels
  * Font Sizes
  * Color Palettes
* Save at least one chart as a PNG file

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## What You Will Learn

* Differences between Matplotlib and Seaborn
* When to use each library
* Chart customization techniques
* Basic data visualization skills in Python
* Creating professional and readable plots
